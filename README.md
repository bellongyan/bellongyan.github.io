## 项目/作品摘要

- in 宁理,校园助手 APP[*Java(Android), MySQL*]

  > 通过 Charles 抓包工具对网络请求进行分析，找到查询接口并使用 OkHttp 发起异步请求，对返回得到的 html 文件使用正则表达式进行有效信息的提取，将教务系统中的数据进行入库并检查是否有更新，然后使其正确显示在应用上，方便快捷的使用该 APP 进行真实校园教务系统信息查询；在字典工具中，使用有道词典和百度翻译的 API 分别进行开发，获取对应的查询接口，使用 OkHttp 网络请求框架对接口发起异步请求，查询对应文档根据内容在获取到的 JSON 对象中，使用 GSON 第三方库将 JSON 对象反序列化成 Java 对象进行使用

- 基于 WinForm 和 SqlServer 技术的 MyQQ 社交软件的设计与实现[*C#, Winform, Sql Server*]

  > 该项目对腾讯 QQ 的在线聊天功能和好友功能进行分析和重新设计，使用 C#的 WinForm 框架实现了基于 SqlServer 数据库技术的 MyQQ 应用。在该项目的数据库表中设计了多个标志位，实现登录记住密码并自动登录和消息是否已读等功能，同时对于查看历史消息功能，采用分页查询提高数据库的查询效率

- 基于 Hadoop 平台的新闻媒体字符统计与分析[*Python 数据抓取, Hadoop 大数据分析*]

  > 使用 python 的爬虫框架对人民网海外版政治、经济、商业、文化、科技等 13 个板块进行新闻数据进行多线程爬取，获取有效新闻 8682 条，将数据进行预处理后存放到分布式存储平台 Hadoop 中，编写 Map/Reduce 代码并生成 jar 包，在 Hadoop 平台进行 Map/Reduce 操作后，并分布式计算结果做出数据统计分析

- 毕设：《基于 HarmonyOS 的校内羽毛球论坛系统的设计与实现》[*前端：ArkUI/ArkTs, 后端：SSM*]

  > 项目基于 HarmonyOS4 的 API9 进行开发，编写原生应用，前端使用 ArkUI/ArkTs 编写应用界面，后端使用 SSM 框架编写，根据项目需求编写对应接口，使用 axios 第三方库对后端接口发起异步网络请求，将得到的数据渲染在应用中，项目数据信息使用 MySQL 数据库进行存储。使用 python 编写爬虫代码，设置定时任务来爬取世界羽联中运动员的相关数据并进行入库操作，针对应用功能和需求进行分析与设计，预期可以实现 HarmonyOS 移动端应用，可以实现用户贴文操作、管理员进行论坛管理、世界羽联运动员排名和用户站内聊天等功能
